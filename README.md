# needtopractice by Teddy Zhang

A conveninent tool for myself to have a tuner, drone, and metronome in the same website. What sets this site apart from other tuning apps is that it intentionally doesn't let you use your phone, and has a more robust drone.

Mostly vibe-coded.

link: needtopractice.com

## Features

- an unforgiving tuner that includes just temperament markings
- a multi-note drone with adjustable tuning
- a metronome with 100+ different click sounds to choose from
- detects if your screen size is too small and tells you to get off of your phone
